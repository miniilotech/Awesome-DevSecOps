# General Info
This repository documents the learning journey of the Cloud DevOps Engineer Expert postgraduate program.

It contains:

•	Training links and resources,

•	Study notes and summaries,

•	Hands-on labs and project deployments.

Purpose:

This project aims to document my learning progress, deepen my understanding of DevOps, and share the story of my postgraduate journey—both the technical milestones and personal experiences along the way.

Why this project?

I've always been curious about the DevOps role. This project helps me explore it as a possible career path. I'm considering DevOps as a direction for my professional development.
